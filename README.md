# social-project-registration-system

# Sistema de Gestão de Inscrições para Projetos Sociais

##  Sobre o Projeto:

Este projeto foi desenvolvido como atividade extensionista do curso de Engenharia de Software, com o objetivo de apoiar a gestão de inscrições de crianças e adolescentes em atividades esportivas oferecidas por um projeto social comunitário.

A solução foi idealizada para centralizar informações dos participantes, facilitar o processo de cadastramento e proporcionar maior organização no controle das atividades realizadas.

##  Problema Identificado:

O processo de inscrição dos participantes era realizado de forma manual, dificultando o controle das informações, o acompanhamento dos alunos inscritos e a organização das atividades esportivas.

##  Solução Proposta:

Foi proposta a criação de um sistema web para realizar:

* Cadastro de participantes;
* Registro de responsáveis legais;
* Armazenamento seguro das informações;
* Consulta rápida de dados;
* Organização das turmas e atividades esportivas;
* Confirmação de inscrição dos participantes.

##  Tecnologias Utilizadas:

* HTML
* CSS
* JavaScript
* Banco de Dados Relacional

##  Arquitetura Proposta

### Front-end

Responsável pela interface de cadastro e consulta de informações.

### Back-end

Responsável pelo processamento das informações e integração com o banco de dados.

### Banco de Dados

Armazenamento dos dados dos participantes e responsáveis.

##  Resultados Esperados

* Redução do tempo de cadastramento;
* Maior organização dos dados;
* Facilidade na consulta de informações;
* Melhor atendimento aos participantes do projeto social.

##  Possíveis Evoluções Futuras:

* Sistema de login para administradores;
* Controle de frequência dos participantes;
* Geração automática de relatórios;
* Notificações por e-mail ou WhatsApp;
* Dashboard com indicadores de participação.

##  Contexto Acadêmico:

Projeto desenvolvido como atividade de extensão universitária, aplicando conhecimentos de programação, banco de dados e desenvolvimento de sistemas para solucionar uma necessidade real da comunidade.
